# 🎓 Student Intelligent Productivity Scheduler

An AI-powered productivity scheduler that predicts your focus level using Machine Learning and generates a personalised study schedule for your day.

**Built by:** Arwa Daaboul — LJMU Computer Science 2026  
**Supervisor:** Dr Hoshang Kolivand

## Features
- 🤖 ML Focus Prediction (Random Forest — 3 models)
- 📅 Time-aware schedules starting from NOW or TOMORROW
- 🆘 Rescue Mode for low-focus days
- 📈 Progress tracking with streaks & trend charts
- 📄 Export to .txt and .ics (Apple/Google Calendar)
